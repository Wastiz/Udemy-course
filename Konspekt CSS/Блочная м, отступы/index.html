При помощи display можно управлять типом тегов. Так я сделал div строчным элементом

Padding является свойством для отступа между контентом и тегом. То есть отступ происходит внутри границ тега.

Margin является свойством для отсутупа между тегами. То есть вне границ элемента

Добавляются отступы для них либо в одной строчке (по часовой), либо отдельными свойствами (margin-top, padding=left)

Для строчных элементов не работает margin top и margin bottom

Если у элементов находящихся рядом два разных маргина, то расстояние между ними будет равнятся большему маргину

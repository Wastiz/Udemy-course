Простой тег для вставки картинок. src - ссылка на картинку. alt - если картинка не прогружается, то высветится значение этого тега

Также атрибутами можно задать высоту и ширину


Этот тег является неким контейнером для картинок. При помощи атрибутов можно задавать при каких размерах экрана

будет появлятся картинка. Картинки могут быть любыми. Главное, чтобы был семантический смысл

является контейнером для картинки и описания картинки

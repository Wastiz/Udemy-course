Главный тег для создания форм, в котором обрабатываются все поля. В него пишутся все элементы для заполнения

Имеет атрибуты action - отвечает за файл, который будет отправлятся с данными формы

Имеет атрибут autocomplete - это когда у пользователя будут варианты заполнения при повторном посещении сайта

Имеет атрибут name, чтобы можно было понять какая это форма при обработке

Тег textarea имеет другой визуальный вид (больше чем у инпут) и имеет перенос строк.

Атрибуты тега textarea:
rows Отвечает за высоту
cols Отвечает за ширину
autofocus Автоматически фокусирует поле, на которое добавляется текст
disabled Делает поле недоступным
readonly Поле остается активным, но его нельзя изменять
placeholder Универсалтьный атрибут. Добавляет текст, который убирается при изменении
maxlength Создает ограничение по символам
required Тоде универсальный атрибут для полей. при этом атрибуте поле обязательно должно ыть заполнено

fieldset - это обводка для формы, что делает UX лучше

legend - это может быть название формы, которая вставляется в обводку fieldset

label - это привязка к инпуту или textarea, при нажатии на label вы можете заполнять форму

select - тег для поля с выпадающим списком




Атрибут multiple дает возможность выбрать сразу несколько вариантов, но пр этом он изменит свою визуальную форму

У optgroup есть два атрибута: disabled и label

Первый блокирует выбор нескольких опций, а второй добавляет label для этих опций

атрибуты option
value Значение опции при отправке на сервер
label указание метки элемента списка В таком случае закрывающий тег не нужен
selected Выбор дефолтного элемента для списка
disabled блокировка для выбора
